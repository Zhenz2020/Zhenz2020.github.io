Application of Data Dimensionality Reduction Methods to Business Data in the Context of Different Time Series Scales

 

1.Introduction

In business analysis, it is often necessary to handle and analyze time series data, but due to objective reasons of data collection, we often encounter time series with different time scales, for example, for GDP data collected quarterly as the dependent variable versus industrial value added collected monthly as the independent variable. In this simple toy model, we need to do some dimensionality reduction measures due to the mismatch of time series. Extending to multivariate data analysis, we need to downscale the size of the time series scale of the independent variable to match the time series scale of the dependent variable.

Similarly for data with consistent time scales, for example, in high-dimensional stock data, the existence of covariance using unprocessed data sets can also make the model over-fitted due to high dimensionality, resulting in lower prediction accuracy, so we also need to achieve data dimensionality reduction for high-dimensional data.

This study mainly aims to study the application of data downscaling in business data, here the downscaling has two parts, the first part is for different time series scale data downscaling to the same scale, the second part is for high-dimensional data, using data downscaling to improve the accuracy of the model. This study conducts a summary comparison of data dimensionality reduction techniques in business analysis by comparing different dimensionality reduction methods with different prediction models.

2.Background

There are various methods for time series dimensionality reduction, such as segmented linear representation, domain transform based representation, model based representation, singular value decomposition representation, etc.

For data dimensionality reduction methods, there are principal component analysis (PCA), linear discriminant analysis (LDA) and ISOMAP based on dimensionality reduction, etc., and there are also Lasso and random forest based on feature selection, etc.

Dimensionality reduction inevitably brings loss of information, and the key is whether the lost data is irrelevant to the dataset. Whether based on feature selection or dimensionality reduction, in time series data, the essence is still to reduce the data noise in business, to get the part of business data in line with the logic of the market.

For data with different time scales, the dimensionality is reduced to the same time scale as the dependent variable, and after the processed high-dimensional data is subjected to variable selection or dimensionality reduction, the obtained data is then put into the prediction model for prediction, which is the logic and background of this study.

As can be seen from the above figure, the RMSEs of all three prediction models are relatively large, indicating that sliding windows are not effective for time series dimensionality reduction. This is also because the sliding window model loses too much information of the time series. The sliding window only collects the weighted average of the values in time, and does not consider the trend and shape of the time series, which loses important information in the time series.

4.3 Model based method to reduce time series dimensionality

4.3.1 Polynomial regression coefficients

We then use polynomial regression coefficients as parameters for time series downscaling, which belongs to the model-based downscaling method, where the regression coefficients of each segment of the data are obtained by polynomial regression of the data, and the regression coefficients of each segment of the data are used as parameters after downscaling.

For this part, we used quadratic polynomial regression and selected 6 months of data as time periods, so each time period returned 3 coefficients.

We first downscaled the time series data according to the above rules, and then in order to see the relationship between the downscaled data and the dependent variable, we clustered the downscaled data, and the clustering method used was k-means, and finally visualized the clustering results, as shown in the figure below.

From the above figure, we can conclude that the categories of clustering classify the dependent variable, and the parameters after polynomial regression of dimensionality reduction do have correlation with the dependent variable.

Similarly for the model prediction of the data after polynomial regression with dimensionality reduction, we get the box plot as below.

Compared with the sliding window method, the RMSE of the polynomial regression dimensionality reduction method is much smaller, and it is also obvious that the RMSE of polynomial regression is also smaller when compared with the direct stretching method which does not produce information loss.

Interestingly, in the polynomial regression dimensionality reduction method, the RMSE of the lasso regression model is smaller than that of the XGBoost model with lasso data dimensionality reduction. Each parameter of the polynomial regression contains the overall characteristics of the time series, so the feature selection after lasso already includes the loss of important information in the time series, which is the reason for the increase of RMSE after data downscaling.
